# Hi, I'm Leopoldo.

I sit at the intersection of literature and technology. As the founder of Grupo Aboio, I've experienced the Brazilian publishing ecosystem firsthand. Now, through [@Aboio Labs](https://github.com/aboio-Labs/), I'm bridging the gap between editorial work and software engineering.

My current mission is to build robust digital tools that help the Brazilian book scene flourish. And I am a huge advocate for functional programming (particularly Gleam âœ¨).

Currently focusing on:

- Developing a custom ERP system tailored for the publishing industry.

- Building platforms to help authors transparently track their book sales.

- Translating complex technical concepts to empower non-technical editorial teams.
